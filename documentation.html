🍔 ما هو النظام؟

نظام طلبات الطعام هو منصة SaaS متعددة المستأجرين (Multi-Tenant) مبنية بـ Laravel، تتيح لأصحاب المطاعم والمتاجر إنشاء متجرهم الخاص بنطاق فرعي مستقل (Subdomain)، وإدارة منتجاتهم وطلباتهم وعملائهم وسائقيهم من لوحة تحكم واحدة متكاملة.


يدعم النظام تطبيق جوال للعملاء لتصفح المنتجات وتقديم الطلبات وتتبعها، وتطبيقاً مستقلاً للسائقين لاستلام الطلبات وتحديث حالتها.

⚙️ التقنيات المستخدمة

Laravel 11 Tenancy for Laravel Laravel Sanctum Laravel Horizon Spatie Permissions Firebase FCM WebSockets Kashier Payment MySQL Redis Laravel Scout REST API

سوبر أدمن
